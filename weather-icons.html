Design Notes

Design philosophy: These icons use a dual-tone filled approach, combining solid filled shapes with reduced-opacity accent elements. This creates visual depth without relying on gradients (which are expensive at 16px) or thin strokes (which disappear at small sizes).


Color palette mapping:

  • #eab308 (amber-500) — Sun, warmth, light mode toggle, lightning
  • #f97316 (orange-500) — Sunrise/sunset warmth, horizon glow
  • #94a3b8 (slate-400) — Clouds, moon, fog, neutral weather
  • #64748b (slate-500) — Storm clouds (darker)
  • #b0bec5 (blue-grey-200) — Overcast layered cloud
  • #60a5fa (blue-400) — Rain, storm rain accent
  • #bfdbfe (blue-200) — Snow dots
  • #ef4444 (red-500) — Gas pump / fuel
  • #cbd5e1 (slate-300) — Subtle cloud highlights

Upgrade differences from current icons:

  • Sun: filled circle center (r=5) instead of stroke-only circle (r=4). More visible at 16px.
  • Clouds: filled path shapes instead of stroke outlines. Reads as solid mass at small sizes.
  • Rain: angled rain lines with drop-shaped terminals. More natural than vertical dashes.
  • Snow: filled circles in cascade pattern instead of dot-points. More visible.
  • Storm: dark filled cloud + bold lightning bolt. Instantly readable.
  • Fog: filled cloud top + fading horizontal lines. More atmospheric than 3 parallel lines.
  • Moon: solid crescent fill instead of stroke outline. Much more readable at 16px.
  • Day Sun: added subtle glow ring (r=6 at 15% opacity). Premium feel.
  • Gas Pump: filled body + window detail + nozzle arm. More recognizable than stroke-only.
  • Sunrise/Sunset: filled semi-circle horizon with arrow indicator. More visual weight.

Icon library references consulted:

  • Tabler Icons (filled variants) — cloud, sun, moon, gas-station, sunrise, sunset shapes
  • Heroicons (solid) — sun ray proportions, filled circle approach
  • Phosphor Icons (duotone) — opacity layering technique for dual-tone feel
  • Basmilius Weather Icons (filled) — cloud shape proportions
